- A practical knowledge loop keeps raw sources, compiled notes, and agent contracts clearly separated.

## 核心结论
1. Agents benefit from structured JSON envelopes for status and execution feedback.
2. Humans benefit from readable Markdown notes and navigation pages.
3. Public repos should ship a small curated sample KB instead of private working data.

## 关键事实 / 证据
- The CLI can return machine-readable maintenance and completion payloads.
- Obsidian-friendly notes can be generated alongside the CLI layer.
- A sample KB makes the public repo understandable without exposing private data.
