<objective>
Build the deterministic patch apply and rollback contract.

Purpose: let agents complete the proposal workflow automatically without hidden LLM calls inside the CLI. Agents may generate patch bundles; Wikify validates, applies, records, and rolls back deterministic text operations.
</objective>

<data_flow>
1. `wikify propose --task-id agent-task-1` writes `sorted/graph-patch-proposals/agent-task-1.json`.
2. A downstream agent reads that proposal and writes a patch bundle JSON:
   - `schema_version: wikify.patch-bundle.v1`
   - `proposal_task_id`
   - `proposal_path`
   - `operations[]`
3. `wikify apply --proposal-path ... --bundle-path ... --dry-run` validates the bundle and reports preflight.
4. `wikify apply --proposal-path ... --bundle-path ...` performs exact deterministic replacements and writes `sorted/graph-patch-applications/<application-id>.json`.
5. `wikify rollback --application-path ... --dry-run` validates rollback.
6. `wikify rollback --application-path ...` restores prior text when current content still matches recorded post-apply hashes.
</data_flow>
