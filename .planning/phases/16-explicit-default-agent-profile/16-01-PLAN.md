<objective>
Build explicit default agent profile shorthand.

Purpose: reduce repeated profile names in automation commands while preserving the explicit external-agent boundary. The user still has to pass `--agent-profile`; the project default only supplies the name.
</objective>

<data_flow>
1. Caller runs `wikify agent-profile --set default --agent-command "python3 agent.py"`.
2. Caller runs `wikify agent-profile --set-default default`.
3. Wikify writes `default_profile: "default"` to `wikify-agent-profiles.json`.
4. Caller runs `wikify maintain-run --agent-profile`.
5. CLI parses bare `--agent-profile` as the default sentinel.
6. Resolver loads the configured default profile and passes its command/timeout into the existing producer path.
7. Calling `wikify maintain-run` without `--agent-profile` still does not run an external producer.
</data_flow>
