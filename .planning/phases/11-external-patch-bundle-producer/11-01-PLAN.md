<objective>
Build an explicit external patch bundle producer adapter.

Purpose: automate the gap between bundle request and patch bundle while keeping provider choice, credentials, and model execution outside hidden CLI defaults. This gives agents a stable way to call Codex/Claude/local scripts/other tools, then uses Wikify's deterministic preflight as the safety boundary.
</objective>

<data_flow>
1. `wikify run-task --id agent-task-1` writes `sorted/graph-patch-bundle-requests/agent-task-1.json` when bundle is missing.
2. `wikify produce-bundle --request-path sorted/graph-patch-bundle-requests/agent-task-1.json --agent-command "<explicit command>"` reads that request.
3. The producer invokes the command with request JSON on stdin and env vars for request/bundle paths.
4. If stdout contains JSON, Wikify writes it to `suggested_bundle_path`; if stdout is empty, Wikify checks whether the command wrote that path.
5. Wikify runs deterministic apply preflight against the produced bundle.
6. The caller runs `wikify run-task --id agent-task-1` again to apply and mark done.
</data_flow>
