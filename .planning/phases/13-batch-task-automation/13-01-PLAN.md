<objective>
Build bounded batch task automation with `wikify run-tasks`.

Purpose: let automation process a small selected set of graph agent tasks without prompting the user, while keeping failure handling, provider execution, and content mutation bounded and auditable.
</objective>

<data_flow>
1. Caller runs `wikify run-tasks --status queued --limit 5 --agent-command "<explicit command>"`.
2. Batch runner loads `sorted/graph-agent-tasks.json`.
3. It selects tasks by status/action/id/limit.
4. It runs each selected task sequentially through `run_agent_task`.
5. Each item records task id, status, result or structured error, and whether execution stopped.
6. Default behavior stops at the first task error. `--continue-on-error` records the error and keeps going.
</data_flow>
