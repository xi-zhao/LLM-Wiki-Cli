<objective>
Build explicit agent command profiles.

Purpose: reduce user interruption and repeated CLI typing by letting projects store named external agent commands, while keeping every semantic generation path explicit, auditable, and outside hidden Wikify provider behavior.
</objective>

<data_flow>
1. Caller runs `wikify agent-profile --set default --agent-command "python3 agent.py" --producer-timeout 120`.
2. Wikify writes `wikify-agent-profiles.json` at the KB root with schema `wikify.agent-profiles.v1`.
3. Caller runs `wikify maintain-run --agent-profile default`.
4. CLI resolves the profile command and timeout before producer execution.
5. Existing `maintain-run -> run-tasks -> run-task -> produce-bundle -> preflight -> apply -> lifecycle` flow runs unchanged.
6. Result metadata exposes that an agent profile was used.
</data_flow>
