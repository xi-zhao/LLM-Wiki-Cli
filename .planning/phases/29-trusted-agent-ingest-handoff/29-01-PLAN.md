# Phase 29 Plan 01: Trusted Agent Ingest Handoff

**Status:** Ready
**Type:** TDD
**Wave:** 1
**Objective:** Make `wikify ingest` produce a trusted-agent work request and agent-friendly completion summary so a calling agent can save, organize, validate, and report knowledge without exposing pipeline internals to humans.

## Requirements

- TAI-01: Product docs describe human natural-language intent and agent-facing `wikify ingest`.
- TAI-02: Successful non-dry-run ingest writes a trusted agent work request artifact.
- TAI-03: Request includes source, content, workspace context, permissions, recovery, page standards, and completion contract.
- TAI-04: Dry-run reports planned request path without writing artifacts.
- TAI-05: Ingest completion summary helps the calling agent produce a human-facing save summary.
- TAI-06: Behavior is tested and documented.

## Tasks

1. Add failing tests for the trusted ingest request path, dry-run planned artifact, request schema fields, and CLI completion summary.
2. Add ingest request artifact helpers and request builder.
3. Wire request writing into `run_ingest`, including failed human-path cases where source capture succeeded.
4. Update CLI completion to expose agent next actions and human-facing summary fields.
5. Update docs to position humans as natural-language requesters and agents as Wikify callers.
6. Run focused tests, full test suite, compile check, and diff check.
7. Write SUMMARY and VERIFICATION artifacts.

## Acceptance Checks

- `wikify ingest --dry-run` returns a planned trusted-agent request path and writes nothing.
- Successful ingest writes `.wikify/ingest/requests/<run-id>.json`.
- The request artifact includes `trusted_agent.permissions.delete_merge: "allowed_with_snapshot"`.
- The request artifact includes source/content/workspace/task/recovery/page_quality/completion sections.
- CLI completion includes `human_summary`, `agent_next_actions`, and request artifact paths.
- Documentation no longer frames normal humans as needing to understand ingest internals.
- `python3 -m unittest tests.test_ingest_pipeline tests.test_wikify_cli -v` passes.
- `python3 -m unittest discover -s tests -v` passes.
- `python3 -m compileall -q wikify` passes.
- `git diff --check` passes.
