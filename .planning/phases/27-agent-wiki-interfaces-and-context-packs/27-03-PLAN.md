<threat_model>
## Threat Model

**Assets:**
- User-facing documentation and protocol reference.
- GSD requirements/state artifacts.
- Verification evidence used to decide whether Phase 27 is complete.

**Threats and mitigations:**
- Misleading product positioning: docs must describe Wikify as a CLI-first wiki artifact generator for humans and agents, not a chat-first RAG app.
- Hidden provider confusion: docs must explicitly say Phase 27 ranking/selection is deterministic and stdlib-only.
- Citation overclaiming: docs must explain explicit citations versus page source-ref fallback and the empty-evidence result.
- Agent integration ambiguity: docs must list exact commands, JSON schema names, and artifact paths.
- False completion: GSD requirement checkboxes and roadmap status change only after tests and smoke workflow pass.
- Protocol drift: `scripts/fokb_protocol.md` must distinguish the new `wikify agent` namespace from existing compatibility commands.
</threat_model>
