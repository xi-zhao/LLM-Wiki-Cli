<objective>
Build a low-interruption graph agent task workflow runner.

Purpose: agents should be able to call one command that advances one queued task as far as deterministic artifacts allow. The runner should create/reuse proposals, detect patch bundles, apply when safe, and update lifecycle state without asking the user.
</objective>

<data_flow>
1. `wikify run-task --id agent-task-1 --dry-run` previews proposal/apply/lifecycle work without writes.
2. `wikify run-task --id agent-task-1` creates/reuses `sorted/graph-patch-proposals/<task-id>.json`.
3. If `sorted/graph-patch-bundles/<task-id>.json` is missing, return `status: waiting_for_patch_bundle` and `next_actions` for an agent.
4. If the patch bundle exists, run deterministic apply.
5. On apply success, mark the task `done` through lifecycle.
6. Return `wikify.agent-task-run.v1` with step statuses and artifacts.
</data_flow>
