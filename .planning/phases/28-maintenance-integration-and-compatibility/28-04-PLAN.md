<threat_model>
## Threat Model

**Assets:** public documentation, protocol docs, GSD completion status, compatibility guarantees, E2E verification evidence.

**Threats and mitigations:**
- False completion: mark requirements complete only after full tests and E2E pass.
- Compatibility regression: CLI tests assert old commands and old task fields still work.
- Hidden provider confusion: docs state maintenance uses explicit agent/verifier commands only.
- Derived artifact misuse: docs say views and agent exports should be regenerated through `wikify views` and `wikify agent export`.
- Evidence gaps: verification artifact records commands, schema versions, task metadata, and preservation rejection results.
</threat_model>
